## ✨ 핵심 기능 (Key Features)

### 1. 긴박감 넘치는 카드 뒤집기 (Card Flip) ❓
- 편성 결과를 바로 공개하지 않고 **물음표 카드** 형태로 보여주어 긴장감을 조성합니다.
- 카드를 클릭할 때마다 멤버 이름이 **에메랄드 빛 효과**와 함께 공개됩니다.
- **'한번에 보기'** 기능을 통해 모든 결과를 0.05초 간격으로 순차 공개할 수 있습니다.

### 2. 티어별 순서 유지 밸런싱 (Balance Mode) ⚖️
- **그룹화 배정:** 팀장급, 일반 멤버 등 그룹별로 명단을 입력하여 실력 밸런스를 맞춥니다.
- **서열 유지 로직:** 먼저 입력한 그룹의 멤버가 결과에서도 **맨 상단**에 위치하도록 설계하여 역할군 구분을 명확히 했습니다.

### 3. PC 최적화 대시보드 레이아웃 (PC Layout) 💻
- **2분할 인터페이스:** PC 환경에서 왼쪽은 '설정', 오른쪽은 '결과'를 나란히 배치해 한눈에 관리 가능합니다.
- **일체형 컨트롤러:** 팀 개수 조절 버튼을 일체형 UI로 디자인하고 숫자를 중앙 정렬하여 조작 편의성을 높였습니다.

## 📝 개발 일지 및 UX 고민

### 1. 사용자 중심의 레이아웃 설계 (UX)
초기에는 위에서 아래로 흐르는 단순 나열 방식이었으나, 실제 사용자가 명단을 입력하며 결과를 동시에 확인하기 불편하다는 점을 발견했습니다. 이를 해결하기 위해 **PC용 2분할 대시보드 레이아웃**으로 전면 수정하여 작업 효율을 높였습니다.

### 2. 디테일한 UI 보정
숫자 입력창의 텍스트가 쏠리는 현상을 잡기 위해 정밀하게 중앙 정렬을 맞추고, `-`, `+` 버튼의 가시성을 높이기 위해 폰트 크기와 두께를 조절하는 등 사소한 디테일이 완성도에 미치는 영향을 경험했습니다.

### 3. 방송 친화적 환경 고려
스트리머들이 저작권 걱정 없이 안심하고 사용할 수 있도록 결과 화면 하단에 사용 가능 안내 문구를 명시했습니다.
